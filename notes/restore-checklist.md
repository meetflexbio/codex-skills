# Restore checklist

## New machine

1. Install Codex.
2. Open this repository.
3. Read `manifest/installed-skills.json`.
4. Reinstall official and community skills from their source links.
5. Copy any custom skill folders from `custom-skills/` to `~/.codex/skills`.
6. Restart Codex.

## Rule of thumb

- Official skills: usually reinstall from source
- Community skills: reinstall from source or freeze a local copy if you depend on a specific version
- Custom skills: keep the full folder in this repo
